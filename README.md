# Project README

## 1. Required Packages and Environment Setup

Before running the project, make sure you have the following installed and configured:

- **Visual Studio 2026**  
- **.NET 10 SDK**  
- **DevExpress .NET Components** (`DevExpressNETComponentsSetup-25.2.3`)  
- **MariaDB**  

Ensure all installations are properly configured and accessible from your system.

---

## 2. Project Dependencies

This project is built using the following dependencies:

- **Blazor** – for building interactive web UI  
- **XPO (eXpress Persistent Objects)** – DevExpress ORM for data management  

Make sure these dependencies are correctly referenced in the project.

---

## 3. How to Run

1. Open the project in **Visual Studio 2026**.  
2. In the toolbar, select **.Blazor.Server and IIS Express** as the run configuration.  
3. Click the **Run (▶)** button to start the application.  

The application should open a page where you can start interacting with it.
